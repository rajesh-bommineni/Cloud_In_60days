# Prework Foundations - Mini Foundations (Days 1–5)
Goal

Build basic comfort with Linux, Git, networking, and cloud concepts so the rest of the roadmap doesn’t feel overwhelming.

Time Commitment

1–2 hours per day, 3–5 days.

What To Expect

By the end of this prework, you should:
	•	Be able to navigate a Linux shell without fear
	•	Understand how Git tracks changes
	•	Have a mental model of IP, DNS, ports
	•	Know what “cloud” means beyond marketing

You do not need to remember every command, just know where things live and what to Google.

Step-by-Step :
1. SDLC Concepts
Focused on three phases : 
    • Requirement Analysis, 
    • Coding & Implmentation
    • Deployment & Maintenance
Review this video here: https://youtu.be/Fi3_BjVzpqk?si=Uhe7GsGvp3QMtUQL

2.	Linux basics (2 sessions)
	•	Open KodeKloud Linux basics: https://kodekloud.com/free-labs/linux
	•	Cover: file system, cd, ls, cat, mkdir, rm, cp, mv, permissions (chmod, chown).
	•	Run every command yourself. Don’t skip the labs.

3.	Linux puzzles (optional but powerful)
	•	Start Bandit:
		https://overthewire.org/wargames/bandit/
	•	Clear first 5–10 levels. Focus on reading the hints and Googling commands you don’t know yet.
	
4.	Git & GitHub basics (1–2 sessions)
	•	Use Learn Git Branching:
		https://learngitbranching.js.org/
	•	Complete the “Introduction Sequence” and at least a couple of branching levels.
	•	Aim to understand: commit, branch, merge, checkout.

5.	Networking basics (1 session)
	•	Take AWS Networking Fundamentals (watch at least the intro/overview sections):
		https://explore.skillbuilder.aws/learn/course/11558/aws-networking-fundamentals
	•	Make notes on: IP, subnet, port, DNS, firewall.
	
6.	Cloud models (1 session)
	•	Watch a short video on public/private/hybrid and IaaS/PaaS/SaaS:
		https://youtu.be/2LaAJq1lB1Q

Checklist
	•	You can navigate a Linux shell and create/delete files
	•	You understand what a Git commit and branch are
	•	You can explain IP, DNS, and port in one sentence each
	•	You know what IaaS, PaaS, SaaS roughly stand for
