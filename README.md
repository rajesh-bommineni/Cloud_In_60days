# Let's Do this!

This repository contains the Foundations track for the 60DaysToCloud program. Each week has its own set of steps to follow; with topics, free resources and notes to follow. Also some mini projects that you can build.

Structure:
- Week1_Cloud_Foundations
- Week2_Linux_Networking
- Week3_Infrastructure_as_Code
- Week4_Containers_Kubernetes
- Week5_CICD_Pipelines
- Week6_Cloud_Security
- Week7_Monitoring_AIOps
- Week8_Final_Project

How to use:
- Each week contains a README with goals, suggested contents, and pointers to labs, examples, and templates..
- Contribute new content by opening a pull request against the chore/add-weekly-folders branch (or main if you prefer).
